# Unity 6.3 Docs

> **Target: Unity 6.3 LTS (6000.3)** — version string nội bộ `6000.3.0f1`. LTS support đến 12/2027.

Personal reference về **settings & optimization** cho Unity 6.3 LTS — tập trung vào cấu hình và tối ưu, code/settings-heavy, beginner-friendly.

## Phạm vi

- **Project type:** 2D hoặc 3D
- **Platform:** Mobile hoặc PC
- Giữ **general** — không đi sâu vào thể loại game cụ thể.

Khác biệt giữa **PC** và **Mobile** được trình bày bằng content tabs ngay trong từng trang; khác biệt **2D vs 3D** được tách section/trang riêng ở nơi thật sự khác (Rendering, Physics, Assets).

## Cách dùng

- **Setup (Quick Start):** checklist settings ngay sau khi tạo project mới.
- **Project Settings / Rendering / Assets / Scripting / Build / Profiling:** tham chiếu theo chủ đề.
- **Snippets:** các đoạn cấu hình / code dùng lại.
